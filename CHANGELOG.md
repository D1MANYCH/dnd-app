# Changelog

Все значимые изменения в проекте **D&D 5e Character Sheet** будут задокументированы в этом файле.

## [2.0.0] - 2024-05-20

### 🚀 Крупные улучшения (Major Features)
- **Модульная архитектура состояния**: Внедрен централизованный менеджер состояний (`state.js`) с реактивной системой обновлений UI.
- **Оптимизация производительности**:
  - Реализован "ленивый" рендеринг (Lazy Rendering) для списка заклинаний.
  - Добавлена виртуализация списков: отрисовка только видимых элементов для работы с 700+ заклинаниями без лагов.
  - Внедрен `debounce` для автосохранения в LocalStorage (задержка 300мс).
- **Доступность (A11y)**:
  - Полная поддержка навигации с клавиатуры.
  - Добавлены ARIA-атрибуты для скринридеров.
  - Улучшена цветовая контрастность интерфейса.

### 🛠 Технические улучшения (Technical Improvements)
- **Синтаксис ES6+**: Полный переход с `var` на `let` и `const` для блочной области видимости.
- **Типизация**: Внедрена строгая типизация через **JSDoc** для всех ключевых функций, интерфейсов и структур данных.
- **Тестирование**:
  - Настроена инфраструктура автотестов на базе **Jest**.
  - Написаны тесты для критической логики (валидация, математика характеристик, поиск, реактивность).
  - Покрытие кода тестами составляет ~85%.

### 🐛 Исправления (Bug Fixes)
- Исправлена ошибка дублирования объявления переменной в анимации кубиков.
- Устранены проблемы рассинхронизации данных и интерфейса при частом редактировании.
- Оптимизирована работа поиска: игнорирование регистра и лишних пробелов.

### 📝 Примечания
- Версия 2.0.0 требует современного браузера с поддержкой ES6 Modules.
- Данные сохраненные в предыдущих версиях совместимы и будут автоматически мигрированы при первом запуске.
