# Аудит названий (REQ-5)

Жалоба: «Названия отличаются от привычных как из официальных книг или с dnd.su».

**Причина.** Базовые наборы переведены по **Fantom Studio PHB 5e** (явно помечено в `data.js`: расы, предыстории, навыки, черты). dnd.su и официальный перевод (Hobby World) местами используют другие термины. Заклинания ранее сверяли с dnd.su только по **данным** (школы, дистанции, концентрация) — **названия не приводили**.

**Источник истины (предлагается):** dnd.su как первичный (самый ходовой в RU-сообществе). Где dnd.su и офиц. перевод расходятся — помечено.

> **Статус (19.06.2026):** эталон = **dnd.su** (решение пользователя). **REQ-5a завершён** — `verifyAllBuilds` 36/36, тесты 242/242 на каждом патче:
> - **Навыки** (v3.28.24): Аркана→Магия, Расследование→Анализ.
> - **Раса/язык** (v3.28.25): Дварф→Дворф, Дварфский→**Дворфийский**; миграция (schema 13).
> - **Предыстории** (v3.28.26): Воин→Солдат, Благородный→Дворянин, Герой народа→Народный герой, Матрос→Моряк, Торговец/Подмастерье→Гильдейский ремесленник; слиты дубли Аколит/Преступник-Шпион/Бродяга (15→11); миграция (schema 14).
> - **Состояния** (v3.28.27): прилагательные формы dnd.su (Ослеплён→Ослеплённый и т.д.), id не тронуты.
>
> **Осталось (по запросу):** 5b заклинания (706) и 5c маг.предметы (193) — сверка имён с dnd.su.

### Ключевая находка разведки: книга 2014 PHantom ≈ dnd.su

Точечная сверка кода ↔ 2014 PHantom PHB ↔ dnd.su показала, что книга **совпадает с dnd.su** почти везде. Смена эталона НЕ обнуляет REQ-5a:

| Навык Investigation | Анализ (REQ-5a) | **Анализ** (327) | Анализ | ✅ совпало |
| Spell Fire Bolt | Огненный болт (Fantom) | **Огненный снаряд** (207) | Огненный снаряд | 🔴 код неверен по обоим — REQ-5b цель подтверждена |
| Состояния | прилагат. (REQ-5a) | Испуганный/Ослеплённый/Невидимый (290,313) | прилагат. | ✅ ~совпало |
| **Раса Dwarf** | **Дварф** (REQ-5a′) | **Дварф** (17) | Дворф | ✅ **РЕШЕНО — откат под книгу (REQ-5a′, v3.28.28)** |

**Вывод:** переключение dnd.su→книги для уже сделанного (REQ-5a) — почти no-op, КРОМЕ точечных конфликтов, где книга расходится с dnd.su. Доказанный — **Дварф/Дворф** (REQ-5a выбрал «Дворф» по dnd.su; книга = «Дварф»).

### Новая неаудированная область: фичи подклассов/классов

Манёвры Боевого мастера ([subclass-choices-data.js:5](../subclass-choices-data.js)) — case study:
