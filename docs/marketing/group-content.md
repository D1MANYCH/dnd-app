# Контент в чате-обсуждении

Модель: **канал = дисциплинированный контент по графику, группа = неформальная кухня**.

## Что писать в группу (vs канал)

| | Канал | Группа |
|---|---|---|
| **Темп** | 3 поста/неделю | Когда есть что сказать |
| **Длина** | Полноценный пост (1500–2500 знаков) | 1–5 строк |
| **Тон** | Структурированный, причёсанный | Личный, как в чате с другом |
| **Скрины** | Готовые, обработанные | Сырые, work-in-progress |
| **Темы** | Релизы, фича-разборы, опросы, FAQ | Мини-апдейты, баги, мысли вслух |
| **Призыв к действию** | «Открой → ссылка» | Не нужно, тут уже все вовлечены |

## Типы сообщений для группы

### 1. Мини-апдейт (короткий)
```
Починил баг с КД у Высшего эльфа в тяжёлой броне. v3.19.5 уже в проде.
```

### 2. WIP-скриншот
```
Работаю над импортом из DnDBeyond. Сложность в том что у них своя схема владений — приходится мапить. Вот пока что выходит [скрин].
```

### 3. Мысль вслух / вопрос
```
Думаю про синхронизацию между устройствами. Варианты:
1) Cloudflare KV — бесплатно, но регистрация
2) Только экспорт/импорт через QR-код (без серверов)
3) Через GitHub Gist как «облако»

Какой подход вам ближе?
```

### 4. Ответ на репорт
```
@username — поправил, обновись (Ctrl+Shift+R чтобы сбросить кеш PWA). Спасибо за репорт!
```

### 5. Контекст к посту канала
```
К сегодняшнему посту в канале про опрос — если хочется обсудить варианты словами, не голосованием, кидайте сюда.
```

### 6. Заметка игрока (без претензии на экспертизу)
```
Играл сегодня варваром. Заметил что в приложении не хватает удобной кнопки «впасть в ярость» — приходится листать. Добавлю на главную панель.
```

## Что НЕ писать в группу

- ❌ Дубли постов из канала (они и так туда приходят автоматически)
- ❌ Длинные посты с заголовками и эмодзи-разделителями (это для канала)
- ❌ Рекламные «у нас 100 подписчиков, спасибо!»
- ❌ Гайды по тактике D&D — не наша экспертиза

## Частота

Норма: 2–5 сообщений в неделю. Главное — естественность, не имитация активности.

Если за день ничего по разработке не сделал — лучше промолчать, чем выдумывать. Это **не** Twitter, тут нет требования постоянного присутствия.

## Когда группа выручает

- Подписчик задал вопрос → отвечаешь развёрнуто (в канале бы такой ответ был лонгридом)
- Сломалось что-то критичное → быстрое объявление без подготовки красивого поста
- Опрос дал неожиданный результат → обсуждение «почему так» сразу в группе
- Кто-то нашёл косяк в гайде билда → обсуждаешь там же, потом фиксишь в коде
