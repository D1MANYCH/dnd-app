Сверяю названия базовых умений классов с Книгой Игрока 2014 («Студия PHantom») — партиями. Прошлой была партия Паладина, теперь пятая: Плут и Монах.

Что изменилось:
🗡️ Плут — по книге: «Воровской архетип» → «Архетип плута», «Мастерство (Expertise)» → «Компетентность», «Надёжный» → «Надёжный талант», «Слепое чувство» → «Слепое зрение», «Скользкий разум» → «Скользкий ум», «Удар мастера» → «Удача». В выборе навыков «Экспертиза» → «Компетентность».

🥋 Монах — по книге: «Очки Ки» → «Ци», «Безоружное движение» → «Движение без доспехов», «Путь монаха» → «Монастырская традиция», «Отклонение снарядов» → «Отражение снарядов», «Замедленное падение» → «Медленное падение», «Оглушающий удар» → «Ошеломляющий удар», «Удары Ки» → «Энергетические удары», «Уклонение» → «Увёртливость», «Безмятежность» → «Спокойствие разума», «Вечная молодость» → «Безвременное тело», «Пустотное тело» → «Пустое тело», «Совершенное тело» → «Совершенство». Энергия монаха в книге называется «ци» — поправил термин в умениях и в трекере ресурсов.

Где название уже совпадало с книгой («Скрытая атака», «Хитрое действие» у Плута; «Боевые искусства», «Шквал ударов», «Чистота тела», «Алмазная душа» у Монаха) — не трогал.

Умения класса показываются по уровню — меняется только отображаемое имя, сохранёнки и готовые билды не затрагиваются (проверка билдов 36/36).
