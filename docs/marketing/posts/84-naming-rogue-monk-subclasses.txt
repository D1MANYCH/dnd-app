📦 DnD-Лист v3.28.53 · 26 июня 2026

Сверяю названия умений подклассов с Книгой Игрока 2014 («Студия PHantom») — партиями. Эта партия — архетипы Плута (Вор, Убийца, Мистический ловкач) и традиции Монаха (Открытая ладонь, Тень, Четыре стихии).

Что изменилось:
🗡 Плут. Вор: «Высшая скрытность» → «Высокая скрытность», «Использование магических устройств» → «Использование магических предметов», «Воровской рефлекс» → «Воровские рефлексы». Убийца: «Бонус убийцы» → «Ликвидация», «Мастер маскировки» → «Дополнительные владения». Мистический ловкач: «Похититель заклинаний» → «Вор заклинаний».

🧘 Монах. Путь открытой ладони: «Открытая ладонь» → «Техники открытой ладони», «Целостность тела» → «Исцеление тела», «Спокойствие» → «Умиротворение». Путь тени: «Теневые искусства» → «Техника тени». Путь четырёх стихий: «Ученик стихий» → «Адепт стихий», «Стихийная настройка» → «Родство со стихией».

🌪 Стихийные дисциплины Монаха: «Клыки огненной змеи» → «Зубы огненной змеи», «Кулак нерушимого воздуха» → «Несокрушимый воздушный кулак», «Натиск духов бури» → «Натиск штормовых духов», «Размашистый удар пепла» → «Испепеляющий удар», «Хватка северного ветра» → «Объятья северного ветра», «Гонг вершины» → «Гонг на вершине горы», «Стойка тумана» → «Туманная стойка», «Оседлать ветер» → «Осёдланный ветер», «Защита вечной горы» → «Прочность вечных гор», «Волна катящейся земли» → «Земляной вал». Термин «ки» → «ци» в описаниях.

Где название уже совпадало с книгой («Быстрые руки», «Мастер проникновения», «Самозванец», «Смертельный удар», «Дрожащая ладонь», «Шаг тени», «Плащ теней», «Оппортунист», «Водяной кнут», «Пламя феникса») — не трогал.

Умения подкласса показываются по уровню — меняется только отображаемое имя, сохранёнки и готовые билды не затрагиваются (проверка билдов 36/36).
