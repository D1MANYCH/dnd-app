Обновление без новых функций: расширено автотестовое покрытие самой рискованной части приложения — импорта/экспорта персонажей. Версия приложения не менялась (v3.25.1).

Изменения:
✅ Автотестов стало 203 (было 191)
🔁 Полный цикл «экспорт → файл → импорт» проверяется автоматически: персонажи (включая мультикласс), история здоровья и добавленные вручную заклинания возвращаются без потерь
🗂 Миграции старых сохранений покрыты тестами: персонажи из ранних версий (языки/инструменты строкой, старые заметки, колдунские пакт-ячейки) корректно достраиваются до текущей схемы
🛡 Отбраковка повреждённых файлов при импорте: битые записи отрезаются, валидные загружаются
🎒 Инвентарь под тестами: слоты по Силе, вес с монетами (50 монет = 1 фунт), мешочки, снятый рюкзак, перенос предметов
🔧 Тех. фикс: страница тестов больше не может затереть данные приложения, если открыта в том же браузере

Тесты гоняются на каждое изменение кода — регресс в сохранности данных теперь ловится до релиза.
