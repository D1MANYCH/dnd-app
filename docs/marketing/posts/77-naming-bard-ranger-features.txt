📦 DnD-Лист v3.28.45 · 23 июня 2026

Сверяю названия базовых умений классов с Книгой Игрока 2014 («Студия PHantom») — партиями. Прошлая пара была Жрец + Друид, теперь третья: Бард и Следопыт.

Что изменилось:
🎵 Бард — 3 умения по книге: «Песня отдыха» → «Песнь отдыха», «Мастерство (Expertise)» → «Компетентность», «Магические секреты» → «Тайны магии».
🏹 Следопыт — 7 умений: «Любимый враг» → «Избранный враг», «Натуралист» → «Исследователь природы», «Осведомлённость о примитивах» → «Первозданная осведомлённость», «Шаг земли» → «Тропами земли», «Укрытие в природе» → «Маскировка на виду», «Чувства дикаря» → «Дикие чувства», «Враг смерти» → «Убийца врагов».

Заодно в списках Следопыта: тип врага «Чудовища» → «Монстры», тип местности «Берег» → «Побережье» — по книге.

Где название уже совпадало с книгой (Бард — «Контрочарование», «Превосходное вдохновение»; Следопыт — «Архетип следопыта», «Исчезновение») — не трогал.

Умения класса показываются по уровню — меняется только отображаемое имя, сохранёнки и готовые билды не затрагиваются (проверка билдов 36/36).
