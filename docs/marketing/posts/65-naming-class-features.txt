📦 DnD-Лист v3.28.33 · 20 июня 2026

Продолжаю сверку названий с Книгой Игрока 2014 («Студия PHantom»). После умений подклассов взялся за умения самих классов — партиями. Первая партия: Варвар, Воин и общие боевые стили.

Что изменилось:
🪓 Варвар: «Путь варвара» → «Путь дикости», «Быстрое движение» → «Быстрое передвижение», «Зверский инстинкт» → «Дикий инстинкт», «Брутальная критика» → «Сильный критический удар», «Неумолимая ярость» → «Непреклонная ярость», «Неукротимый» → «Дикий чемпион».
⚔️ Воин: «Стиль боя» → «Боевой стиль», «Боевой порыв» → «Всплеск действий», «Неумолимый» → «Упорный».
🛡️ Боевые стили (Воин/Паладин/Следопыт): «Двуручный бой» → «Сражение большим оружием», «Защитник» → «Оборона», «Двуоружный бой» → «Сражение двумя оружиями».

Где название уже совпадало с книгой (Ярость, Защита без доспехов, Безрассудная атака, Чувство опасности, Дополнительная атака; стили Стрельба, Защита, Дуэлянт) — не трогал.

На сохранёнках и готовых билдах не сказывается: имена умений только отображаются, в персонажах не хранятся. Проверка билдов 36/36.
