Сверяю названия умений подклассов с Книгой Игрока 2014 («Студия PHantom») — партиями. Эта партия — домены Жреца (7 доменов из Книги Игрока: Жизни, Света, Войны, Знаний, Природы, Бури, Обмана).

Что изменилось:
⛪ Префикс умений канала приведён к книге: «Канал: …» → «Божественный канал: …» во всех доменах.

✨ Имена фич по книге, например: «Ученик жизни» → «Поборник жизни», «Заговор Свет» → «Дополнительный заговор», «Защитная вспышка» → «Защищающая вспышка», «Аура солнца» → «Корона света», «Боевой жрец» → «Боевой священник», «Аватар битвы» → «Воплощение битвы», «Мощный заговор» → «Могущественное колдовство», «Покорение зверей и растений» → «Очарование животных и растений», «Укрощение стихий» → «Сдерживание стихий», «Повелитель природы» → «Мастер природы», «Громовой удар» → «Удар грома», «Дитя бури» → «Рождённый бурей», «Мантия двойника» → «Двуличность», «Мантия теней» → «Плащ теней», «Улучшенный двойник» → «Улучшенная двуличность».

Где название уже совпадало с книгой («Благословенный целитель», «Высшее исцеление», «Благословение знаний», «Видения прошлого», «Улучшенная вспышка» и др.) — не трогал.

Умения подкласса показываются по уровню — меняется только отображаемое имя, сохранёнки и готовые билды не затрагиваются (проверка билдов 36/36).
