Сверяю названия базовых умений классов с Книгой Игрока 2014 («Студия PHantom») — партиями. Прошлой была партия Чародея и Волшебника, теперь седьмая, последний из заклинателей: Колдун.

Что изменилось:
🕯️ Базовые умения по книге: «Покровитель» → «Потусторонний покровитель», «Пакт-магия» → «Магия договора», «Мистические воззвания» → «Таинственные воззвания», «Пакт» → «Предмет договора», «Мистическое арканум» → «Таинственный арканум», «Тёмный господин» → «Таинственный мастер». Термин «пакт» в умениях заменён на «договор».

🔗 Дары договора по книге: «Пакт Цепи/Клинка/Фолианта» → «Договор цепи/клинка/гримуара».

👁️ Воззвания — 25 из 32 по книге: например «Болезненная вспышка» → «Мучительный взрыв», «Зрение дьявола» → «Дьявольский взгляд», «Цепи Кокитуса» → «Цепи Карцери», «Ведьмино зрение» → «Ведьмачий взор», «Похититель судеб» → «Вор пяти судеб».

Где название уже совпадало с книгой («Доспех теней», «Жаждущий клинок», «Маска многих лиц», «Туманные видения» и др.) — не трогал.

Умения класса показываются по уровню — меняется только отображаемое имя, сохранёнки и готовые билды не затрагиваются (проверка билдов 36/36).
