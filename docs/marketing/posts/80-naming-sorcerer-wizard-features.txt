📦 DnD-Лист v3.28.48 · 23 июня 2026

Сверяю названия базовых умений классов с Книгой Игрока 2014 («Студия PHantom») — партиями. Прошлой была партия Плута и Монаха, теперь шестая, заклинатели: Чародей и Волшебник.

Что изменилось:
🔮 Чародей — по книге: «Магическое происхождение» → «Происхождение чародея», «Очки чародея» → «Источник магии». Энергия чародея в книге называется «единицы чародейства» — поправил термин в умениях, в трекере ресурсов и в метамагии. Метамагия по книге: «Бережное» → «Аккуратное», «Дальнобойное» → «Далёкое», «Возвышенное» → «Непреодолимое», «Незримое» → «Неуловимое», «Сдвоенное» → «Удвоенное».

📖 Волшебник — по книге: «Традиция волшебства» → «Магическая традиция», «Мастерство заклинаний» → «Мастерство заклинателя», «Именные заклинания» → «Фирменное заклинание».

Где название уже совпадало с книгой («Метамагия», «Усиленное/Продлённое/Ускоренное заклинание» у Чародея; «Книга заклинаний», «Магическое восстановление» у Волшебника) — не трогал.

Умения класса показываются по уровню — меняется только отображаемое имя, сохранёнки и готовые билды не затрагиваются (проверка билдов 36/36).
