# Пост для DTF / VC — как опубликовать

Готовый комплект для статьи-анонса на DTF (раздел «Настолки») или VC.ru.

## Что в папке

| Файл | Что это |
|---|---|
| `post-dtf-vc.md` | **Текст поста** — копипастишь в редактор DTF/VC. Внутри помечены места под картинки. |
| `01-cover.png` | **Обложка поста** (1280×720). Ставится сверху как cover. |
| `02-features.png` | Картинка «что внутри» — вставляется в раздел «Что внутри». |
| `preview.html` | **Превью целиком** — открой в браузере, чтобы увидеть, как пост выглядит. Нужен PDF? Здесь же: `Ctrl+P` → «Сохранить как PDF». |
| `gen_promo.py` | Скрипт, которым сгенерированы PNG (если захочешь поправить текст/цвета — `python gen_promo.py`). |

## Как выложить (3 шага)

1. **Текст.** Открой `post-dtf-vc.md`, скопируй текст (без строк, начинающихся с `> 🖼️` — это пометки для тебя), вставь в редактор DTF/VC. Заголовки `##` редактор сам распознает как подзаголовки.
2. **Обложка.** Загрузи `01-cover.png` как обложку статьи (DTF: «Добавить обложку»; VC: первый блок-картинка).
3. **Картинка в текст.** В разделе «Что внутри» добавь блок-картинку и загрузи `02-features.png`.

> На DTF и VC картинки **загружаются файлами в редактор** — PDF/markdown туда не вставляется с изображениями, поэтому отдаю текст и PNG отдельно.

## Живые скриншоты интерфейса (опционально, но сильно усиливает пост)

Промо-картинки — это оформление. Читатели DTF/VC любят видеть **реальный интерфейс**. 3–4 живых скрина добавят доверия. Снять по официальному воркфлоу — см. [../../screenshots/README.md](../../screenshots/README.md) (Chrome → F12 → Device Mode 1440×900 → `Ctrl+Shift+P` → `Capture screenshot`).

**Шот-лист (что снять и как дойти до экрана):**

1. **Лист персонажа.** Открыть приложение → закрыть приветствие → `📘 По готовому билду` → выбрать билд → персонаж создан, ты на вкладке «📋 Лист персонажа». Для богатого вида подними уровень до 5–10. → скрин.
2. **Выбор готового билда** (показывает «36 билдов»). На стартовом экране `📘 По готовому билду` → открылась модалка с карточками билдов. → скрин.
3. **Заклинания** (показывает «719 заклинаний»). Нужен персонаж-заклинатель (билд мага/жреца/барда) → вкладка «✨ Заклинания» → открой карточку заклинания. → скрин.
4. *(бонус)* **3D-кубики.** На листе нажми быстрый бросок `d20` справа → дождись анимации кубика. → скрин.
5. *(бонус)* **Бой.** Вкладка «⚔️ Бой» с инициативой/состояниями. → скрин.

Тема — **тёмная** (родная). Темплейт имён файлов и конвертацию в WebP см. в том же `screenshots/README.md`.
